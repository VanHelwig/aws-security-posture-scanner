# AWS Security Posture Scanner

AWS security posture scanning application built with Python, boto3, and Terraform.

## Development Setup

Create and activate a virtual environment:

```bash
python -m venv .venv
source .venv/bin/activate
```

Install the project in editable mode:

```bash
pip install -e .
```

## Run the Application

```bash
posture-scanner
```

Expected output:

```text
AWS Security Posture Scanner MVP
AWS Region: us-east-1
```
